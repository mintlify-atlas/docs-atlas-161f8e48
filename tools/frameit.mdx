## Overview

frameit allows you to quickly put your app screenshots into device frames with optional custom text and backgrounds. This makes your screenshots look more professional for the App Store and marketing materials.

<Info>
frameit works seamlessly with screenshots generated by [snapshot](/tools/snapshot) and [screengrab](/tools/screengrab).
</Info>

## Installation

frameit is part of fastlane and requires ImageMagick:

```bash
brew install imagemagick
```

## Quick Start

<Steps>
  <Step title="Download device frames">
    ```bash
    fastlane frameit download_frames
    ```
  </Step>
  
  <Step title="Run frameit on your screenshots">
    Navigate to your screenshots directory and run:
    
    ```bash
    fastlane frameit
    ```
    
    Or specify a path:
    
    ```bash
    fastlane frameit silver /path/to/screenshots
    ```
  </Step>
</Steps>

## Configuration

Create a `Framefile.json` in your screenshots directory to add custom text and styling:

```json
{
  "device_frame_version": "latest",
  "default": {
    "title": {
      "color": "#FFFFFF",
      "font": "./fonts/MyFont-Bold.ttf"
    },
    "background": "./backgrounds/background.jpg",
    "padding": 50,
    "show_complete_frame": false
  },
  "data": [
    {
      "filter": "01Login",
      "title": {
        "text": "Sign in with ease"
      }
    },
    {
      "filter": "02MainFeed",
      "title": {
        "text": "Discover amazing content"
      }
    }
  ]
}
```

## Key Options

<ParamField path="white" type="boolean" default="false">
  Use white/silver device frames.
  
  ```ruby
  frameit(white: true)
  ```
</ParamField>

<ParamField path="silver" type="boolean" default="false">
  Use silver device frames (alias for white).
  
  ```ruby
  frameit(silver: true)
  ```
</ParamField>

<ParamField path="rose_gold" type="boolean" default="false">
  Use rose gold device frames.
  
  ```ruby
  frameit(rose_gold: true)
  ```
</ParamField>

<ParamField path="gold" type="boolean" default="false">
  Use gold device frames.
  
  ```ruby
  frameit(gold: true)
  ```
</ParamField>

<ParamField path="force_device_type" type="string">
  Force a specific device type. Useful for Mac screenshots where sizes vary.
  
  ```ruby
  frameit(force_device_type: "iPhone 15 Pro")
  ```
</ParamField>

<ParamField path="use_platform" type="string" default="ios">
  Choose a platform: `IOS`, `ANDROID`, or `ANY`.
  
  ```ruby
  frameit(use_platform: "ANDROID")
  ```
</ParamField>

<ParamField path="resume" type="boolean" default="false">
  Resume frameit instead of reprocessing all screenshots.
  
  ```ruby
  frameit(resume: true)
  ```
</ParamField>

## Advanced Framefile Configuration

### Title Styling

```json
{
  "default": {
    "title": {
      "color": "#FFFFFF",
      "font": "./fonts/MyFont-Bold.ttf",
      "font_size": 48,
      "line_spacing": 10,
      "text_alignment": "center"
    }
  }
}
```

### Background Images

```json
{
  "default": {
    "background": "./backgrounds/gradient.jpg",
    "padding": 50
  }
}
```

### Per-Screenshot Configuration

```json
{
  "data": [
    {
      "filter": "01Login",
      "title": {
        "text": "Sign in with ease",
        "color": "#4A90E2"
      },
      "background": "./backgrounds/login_bg.jpg"
    },
    {
      "filter": "02MainFeed",
      "title": {
        "text": "Discover content",
        "color": "#F5A623"
      },
      "background": "./backgrounds/feed_bg.jpg"
    }
  ]
}
```

### Keyword Substitution

frameit supports dynamic text substitution:

- `{platform}` - Platform name (iOS, Android)
- `{version}` - App version
- `{build}` - Build number

```json
{
  "title": {
    "text": "Version {version} ({build})"
  }
}
```

## Tips & Best Practices

<Tip>
**Use high-quality backgrounds**: For best results, use high-resolution background images that match your brand.
</Tip>

<Warning>
**Font files**: Make sure to include the font files in your project if you're using custom fonts.
</Warning>

<Tip>
**Test different colors**: Try different frame colors to see which best matches your app's design.
</Tip>

<Check>
**Download frames first**: Always run `fastlane frameit download_frames` before using frameit for the first time.
</Check>

## Troubleshooting

### ImageMagick not found

Install ImageMagick:

```bash
brew install imagemagick
```

### Device frames not found

Download the latest device frames:

```bash
fastlane frameit download_frames
```

### Incorrect device detection

Force a specific device type:

```ruby
frameit(force_device_type: "iPhone 15 Pro")
```
